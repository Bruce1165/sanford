## 📝 参数说明

### 温和上涨阶段细分说明

#### 震荡期
- **目的**: 杯体形成的主要阶段，价格在杯底和杯沿之间震荡
- **特征**:
  - 有涨有跌
  - 不能跌过杯底价格（由 `OSCILLATE_PRICE_FLOOR_PCT` 控制）
  - 不能太过于接近杯沿价格（由 `OSCILLATE_PRICE_CEIL_PCT` 控制）
- **典型持续**: 15-30天

#### 快速上涨期
- **目的**: 杯体完成的最后阶段，价格加速上涨至右侧杯沿
- **特征**:
  - 持续13-15天
  - 价格上涨增速（由 `RAPID_ASCENT_ACCELERATION` 控制加速度）
  - 直至达到右侧杯沿价格
- **典型持续**: 13-15天

## 🔧 参数调整建议

### 保守型策略
```json
{
  "RIM_INTERVAL_MIN": 50,
  "CUP_DEPTH_MIN": 0.10,
  "HANDLE_MAX_DROP_PCT": 0.3,
  "VOLUME_RATIO_THRESHOLD": 1.2
}
```
- 提高形态质量要求
- 降低假阳性率
- 适合稳健投资者

### 激进型策略
```json
{
  "RIM_INTERVAL_MIN": 35,
  "CUP_DEPTH_MIN": 0.03,
  "HANDLE_MAX_DROP_PCT": 0.6,
  "VOLUME_RATIO_THRESHOLD": 0.9
}
```
- 放宽形态要求
- 增加潜在机会
- 适合积极投资者

## 📊 参数影响分析

| 参数 | 增加值 | 减少值 | 影响 |
|-----|-------|-------|------|
| `RIM_INTERVAL_MIN` | 减少筛选结果 | 增加筛选结果 | 形态形成时间要求 |
| `CUP_DEPTH_MIN` | 减少浅杯 | 增加浅杯 | 杯深要求 |
| `ASCENT_RAPID_MIN` | 延长快速上涨 | 缩短快速上涨 | 上涨节奏要求 |
| `HANDLE_MAX_DROP_PCT` | 允许更大下跌 | 限制下跌 | 杯柄形态宽松度 |
| `VOLUME_RATIO_THRESHOLD` | 减少结果 | 增加结果 | 成交量验证严格度 |
