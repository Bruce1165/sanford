## 欧奈尔杯柄形态 (CANSLIM) 选股系统

---

## 一、系统概述

### 1.1 策略来源

本系统基于**威廉·欧奈尔 (William J. O'Neil)** 的 CANSLIM 投资体系，特别是其中的杯柄形态 (Cup and Handle Pattern) 选股法则。欧奈尔通过对 1880-2008 年间美国股市的深入研究，总结出这一高胜率的中长线形态识别方法。

### 1.2 核心思想

杯柄形态代表股票的**筑底→洗盘→突破**完整周期：
- **杯体 (Cup)**：股价下跌后逐步回升，形成U型底部
- **杯柄 (Handle)**：在杯体右侧高点附近小幅回调，清洗浮筹
- **突破 (Breakout)**：放量突破柄部高点，开启主升浪

### 1.3 系统特点

| 特性 | 说明 |
|------|------|
| **标准化** | 严格遵循欧奈尔原版参数 |
| **可视化** | Dashboard 直观展示筛选结果 |
| **可验证** | 支持单只股票即时检查 |
| **风控集成** | 自动生成止损位和目标位 |

**标准参数：**

| 参数 | 取值 | 说明 |
|------|------|------|
| **杯体周期** | 50-60 天 | 2.5-3个月形成完整杯体 |
| **杯深范围** | 12% - 35% | 回调幅度 |
| **杯型要求** | U 型 | 最低点在 30%-70% 中部区域 |
| **排除 V 型** | <30% 或 >70% | 尖锐底部视为假形态 |

                       │

**标准参数：**

| 参数 | 取值 | 说明 |
|------|------|------|
| **柄部周期** | 15-20 天 | 3-4周形成柄部 |
| **柄调幅度** | 5% - 12% | 回调幅度 |
| **成交量** | <85% | 柄部缩量（洗盘特征） |
| **突破确认** | >柄部高点 | 收盘价突破 |

## 五、风控体系

### 5.1 自动风控计算

**计算公式：**

```
止损位 = 柄部最低点 × 0.97

目标位1 = 突破价格 + 杯体深度
目标位2 = 突破价格 + 1.5 × 杯体深度

盈亏比 = (目标位 - 突破价) / (突破价 - 止损位)
```

### 5.2 风险控制原则

| 原则 | 说明 |
|------|------|
| **硬性止损** | 跌破止损位立即出场，不犹豫 |
| **移动止盈** | 浮盈 >20% 后，以 10 日均线作为跟踪止损 |
| **时间止损** | 10 天未达目标 1，减半仓；20 天强制清仓 |
| **仓位管理** | 单只股票不超过总仓位 20% |

## 八、常见问题

### Q1: 为什么选出的股票这么少？
**A:** 欧奈尔标准本身就很严格。震荡市参数下，每天通常只有 0-5 只股票符合。这恰恰说明系统的有效性——只选最优质的形态。

### Q2: 杯深超过 35% 可以吗？
**A:** 不建议。杯深 >35% 说明抛压太重，即使突破也容易被套牢盘打压。欧奈尔研究发现 12%-35% 是最理想的区间。

### Q3: 为什么必须 MA200 向上？
**A:** MA200 代表长期趋势。如果 MA200 向下，说明大盘或个股处于长期熊市，此时突破成功率极低。

### Q4: RS 分数怎么计算？
**A:** 基于过去 12 个月涨幅相对于市场的排名。RS≥85 表示该股票超越了 85% 的个股，属于强势股。
