# 老鸭头数据库分类器 (Lao Ya Tou Database Classifier)

## 概述

老鸭头数据库分类器是一个基于SQLite数据库直接查询和筛选的老鸭头模式股票分类工具。该工具实现了老鸭头技术分析的三种信号分类系统，能够从5033只A股股票中自动识别符合老鸭头模式的股票，并根据不同的入场信号进行分类。

**创建日期**: 2026-04-17
**状态**: ✅ 已完成并验证
**相关文件**:
- `scripts/lao_ya_tou_db_classifier.py` - 基础版分类器
- `scripts/lao_ya_tou_db_classifier_enhanced.py` - 增强版分类器
- `scripts/analyze_lyt_results.py` - 结果分析工具

---

## 功能特性

### 核心功能

1. **直接数据库查询**
   - 直接从SQLite数据库读取日线数据
   - 自动转换为周线数据进行分析
   - 避免数据重复加载和处理

2. **三种信号分类**
   - **信号1（鸭鼻孔金叉）**: 激进入场信号
   - **信号2（鸭嘴金叉）**: 稳健入场信号
   - **信号3（放量突破）**: 追涨买入信号

3. **灵活筛选模式**
   - **严格模式**: 使用标准老鸭头参数（振幅≥65%，空隙2-10%）
   - **宽松模式**: 使用放宽参数（振幅≥50%，空隙1-15%）

4. **置信度评分**
   - 基于MA空隙大小
   - 成交量比率分析
   - 趋势强度评估
   - 支撑位距离计算

5. **多格式输出**
   - JSON格式：程序化处理
   - Excel格式：人工分析和可视化
   - 控制台输出：快速查看结果

## 技术架构

### 数据流

```
SQLite Database (stock_data.db)
    ↓
daily_prices 表（日线数据）
    ↓
转换为周线数据（按周聚合）
    ↓
计算技术指标（MA5/MA10/MA30, 量比等）
    ↓
检测MA交叉（金叉/死叉）
    ↓
分析老鸭头模式
    ↓
分类为三种信号
    ↓
输出结果（JSON + Excel）
```

### 核心算法

#### 1. 周线转换

```python
# 按周聚合，取每周最后一个交易日
df['week_key'] = df['year'].astype(str) + '_' + df['week_num'].astype(str)
weekly_df = df.groupby('week_key').last().reset_index()
```

#### 2. 技术指标计算

```python
# 移动平均线
df['ma5'] = df['close'].rolling(window=5).mean()
df['ma10'] = df['close'].rolling(window=10).mean()
df['ma30'] = df['close'].rolling(window=30).mean()

# MA空隙
df['ma_gap'] = ((df['ma5'] - df['ma10']) / df['ma10'] * 100)

# 量比
df['volume_ma5'] = df['volume'].rolling(window=5).mean()
df['volume_ratio'] = df['volume'] / df['volume_ma5']
```

#### 3. 交叉检测

```python
# 金叉检测：MA5从下向上穿过MA10
if ma5_curr > ma10_curr and ma5_prev <= ma10_prev:
    golden_cross = True

# 死叉检测：MA5从上向下穿过MA10
if ma5_curr < ma10_curr and ma5_prev >= ma10_prev:
    death_cross = True
```

#### 4. 模式识别

```python
# 老鸭头模式要素：
# 1. 死叉后出现金叉
# 2. MA5 > MA10 > MA30（多头排列）
# 3. 价格振幅符合要求
# 4. MA空隙在合理范围
# 5. 回踩MA30支撑
```

## 信号分类标准

### 信号1：鸭鼻孔金叉（激进）

**触发条件**:
1. 死叉后出现第一次金叉
2. MA5快速金叉MA10（within 2-3周）
3. 金叉时空隙较小（1-10%）
4. MA5 > MA10 > MA30（多头排列）
5. 回踩30周线支撑有效

**参数配置**:
- 振幅要求: ≥50%（宽松模式） / ≥65%（严格模式）
- MA空隙: 1-15%（宽松模式） / 2-10%（严格模式）
- 回踩窗口: 12周（宽松模式） / 10周（严格模式）

**仓位建议**: 3-4成
**风险级别**: 中高（金叉后的假突破风险）
**止损位**: 30周线下方3%，或跌破金叉当周K线实体下沿

**筛选结果（2026-04-16）**:
- 宽松模式: 126只股票
- 严格模式: 2只股票
- Top 5高置信度: 深华发A(000020)、福晶科技(002222)、中利集团(002309)、中瓷电子(003031)、中际旭创(300308)

### 信号2：鸭嘴金叉（稳健）

**触发条件**:
1. 死叉后出现第二次或更多次金叉
2. MA5/MA10开口向上（空隙增大）
3. MA5 > MA10 > MA30（多头排列巩固）
4. 成交量温和放大

**参数配置**:
- 金叉次数: ≥2次
- 空隙增大: 后一次金叉空隙 > 前一次
- 成交量: 量比 > 1.0

**仓位建议**: 4-5成
**风险级别**: 中等（趋势确认后入场）
**止损位**: 30周线下方5%

**筛选结果（2026-04-16）**:
- 宽松模式: 0只股票
- 严格模式: 0只股票
- **说明**: 当前市场环境下，符合鸭嘴金叉条件的股票较少

### 信号3：放量突破（追买）

**触发条件**:
1. 出现金叉信号（信号1或信号2）
2. 成交量显著放大（量比 > 1.5）
3. 价格突破或接近鸭头顶（近期高点）
4. MA5/MA10加速向上

**参数配置**:
- 量比要求: >1.3（宽松模式） / >1.5（严格模式）
- 价格突破: ≥近期高点的95%
- 趋势确认: MA5、MA10、MA30均向上

**仓位建议**: 2-3成（追涨需谨慎）
**风险级别**: 高（高位追涨，需严格止损）
**止损位**: 突破位下方3-5%

**筛选结果（2026-04-16）**:
- 宽松模式: 2只股票（滨海能源000695、宇环数控002903）
- 严格模式: 0只股票

### 置信度计算

**信号1置信度**:
```python
# 基于MA空隙（空隙越小，置信度越高）
gap_score = 1.0 - (gap / max_gap)
confidence = gap_score * 0.7 + 0.3  # 基础置信度
```

**信号2置信度**:
```python
# 基于空隙改善幅度
gap_improvement = (current_gap - prev_gap) / prev_gap
confidence = min(0.6 + gap_improvement * 2.0, 0.95)
```

**信号3置信度**:
```python
# 基于量比
confidence = 0.5 + min(volume_ratio / 3.0, 0.45)
```

## 注意事项

### 数据要求

1. **最小数据量**: 至少52个交易日（约1年）
2. **数据完整性**: 确保日线数据连续无缺失
3. **数据时效性**: 建议使用最新交易日数据

### 筛选限制

1. **市场环境**: 不同市场环境下，符合条件的股票数量差异很大
2. **参数敏感**: 严格模式可能筛选出极少股票，宽松模式可能包含较多噪音
3. **信号时效**: 信号日期与筛选日期可能存在时间差

### 风险提示

1. **技术分析局限**: 基于历史数据，不能保证未来表现
2. **假信号风险**: 金叉后可能再次死叉（假突破）
3. **市场风险**: 需结合市场整体环境判断
4. **止损重要性**: 严格执行止损，控制风险
