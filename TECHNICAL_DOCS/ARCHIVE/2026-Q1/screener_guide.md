## 2. 筛选器详细说明

### 2.1 咖啡杯形态 (coffee_cup_screener)

**形态描述**: 欧奈尔经典杯柄形态，杯身底部圆润，右侧杯柄缩量回调

**识别条件**:
1. 杯身回调幅度 15%-33%
2. 杯身形成时间 7-65周
3. 杯柄位于杯身上半部分
4. 杯柄回调幅度 < 15%
5. 杯柄成交量萎缩

**输出字段**:
```
- 股票代码/名称
- 当前价格/涨跌幅/换手率
- 杯柄日期/杯柄价格
- 杯沿价格/价格差%
- 杯深%/间隔天数
- 上涨原因( LLM分析 )
- 行业分类/相关概念
```

**使用示例**:
```bash
# 手动运行
python3 scripts/coffee_cup_screener.py --date 2026-03-18

# 查看结果
open data/screeners/coffee_cup/2026-03-18.xlsx
```

---

### 2.2 涨停金凤凰 (jin_feng_huang_screener)

**形态描述**: 涨停后横盘整理，缩量不补缺口，再次放量突破

**识别条件**:
1. 涨停板(涨幅≥9.8%)
2. 留下跳空缺口
3. 横盘3-5天
4. 横盘期缩量至涨停日50%以下
5. 再次放量突破(≥1.5倍)

**可调参数**:
```bash
--min-days 3        # 最小横盘天数
--max-days 5        # 最大横盘天数
--max-pullback 0.0  # 最大回调幅度
--shrink-threshold 0.5  # 缩量阈值
--breakout-ratio 1.5    # 突破放量倍数
```

---

### 2.3 涨停银凤凰 (yin_feng_huang_screener)

**形态描述**: 涨停后缩量回调，但未回补缺口，再次放量上涨

**识别条件**:
1. 涨停板(涨幅≥9.8%)
2. 留下跳空缺口
3. 缩量回调2-7天
4. 回调幅度 < 50%
5. 再次放量突破

**与金凤凰区别**: 银凤凰允许回调幅度更大，不强制横盘

---

### 2.4 涨停试盘线 (shi_pan_xian_screener)

**形态描述**: 低位横盘后拉出放量阳线试盘，随后缩量回调，再次放量启动

**识别条件**:
1. 20日低位横盘(涨幅<10%)
2. 拉出放量阳线(可能是假阳线)
3. 随后缩量回调
4. 再次放量突破试盘线高点

---

### 2.5 二板回调 (er_ban_hui_tiao_screener)

**形态描述**: 连续两个涨停板后缩量回调

**识别条件**:
1. 首板涨停
2. 次日二板涨停
3. 二板后缩量回调
4. 回调不破首板最低价

**输出字段**:
```
- 二板日期/距今天数
- 首板最低价/二板最高价
- 支撑位距离%
- 回调状态(缩量/回踩/突破)
```

---

### 2.6 涨停倍量阴 (zhang_ting_bei_liang_yin_screener)

**形态描述**: 涨停后次日高开低走放量阴线，随后缩量洗盘

**识别条件**:
1. 涨停板
2. 次日倍量阴线(成交量>涨停日2倍)
3. 随后缩量至地量
4. 地量后再次放量上涨

**可调参数**:
```bash
--limit-days 21     # 回看时间范围(交易日)
```

---

### 2.7 20日突破 (breakout_20day_screener)

**形态描述**: 股价突破20日新高，成交量放大

**识别条件**:
1. 收盘价创20日新高
2. 成交量大于5日均量
3. 涨幅适中(避免追高)

**特点**: 最简单的突破筛选器，适合快速扫描

---

### 2.8 主升突破 (breakout_main_screener)

**形态描述**: 主升浪中的突破形态，结合多周期均线

**识别条件**:
1. 股价在主要均线上方
2. 突破近期整理平台
3. 成交量配合放大
4. 趋势向上确认

---

### 2.9 每日热冷股 (daily_hot_cold_screener)

**功能描述**: 统计每日市场热点和冷门板块

**输出内容**:
```
🔥 热门板块:
  - 板块名称/上涨家数/涨停家数/平均涨幅

❄️ 冷门板块:
  - 板块名称/下跌家数/跌停家数/平均跌幅

📊 市场情绪指标:
  - 涨跌家数比
  - 涨停/跌停比
  - 成交量变化
```

**使用方式**:
```bash
# 盘后市场分析
python3 scripts/daily_hot_cold_screener.py
```

---

### 2.10 双收板 (shuang_shou_ban_screener)

**形态描述**: 收盘价等于涨停价的涨停板(无影线)

**识别条件**:
1. 收盘价 = 涨停价
2. 开盘价 = 涨停价 或 开盘价 < 涨停价但快速封板
3. 全天无打开

---

### 2.11 A股21选股 (ashare_21_screener)

**功能描述**: 综合多因子的选股系统，整合多种筛选逻辑

**选股维度**:
1. 趋势强度
2. 成交量配合
3. 资金流向
4. 技术指标共振
5. 基本面筛选

## 3. 运行方式

### 3.1 手动运行

#### 单筛选器运行

```bash
# 基本语法
python3 scripts/{screener_name}.py --date YYYY-MM-DD

# 示例: 运行咖啡杯筛选器
python3 scripts/coffee_cup_screener.py --date 2026-03-18

# 示例: 运行金凤凰(带参数)
python3 scripts/jin_feng_huang_screener.py --date 2026-03-18 --min-days 2 --max-days 6
```

#### 通用参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `--date` | 指定分析日期 | `--date 2026-03-18` |
| `--no-news` | 跳过新闻抓取 | `--no-news` |
| `--no-llm` | 跳过LLM分析 | `--no-llm` |
| `--no-progress` | 禁用进度跟踪 | `--no-progress` |
| `--restart` | 强制重新开始 | `--restart` |
| `--db-path` | 指定数据库路径 | `--db-path data/stock_data.db` |

### 3.2 批量运行

```bash
# 运行所有筛选器
python3 scripts/run_all_screeners.py --date 2026-03-18

# 运行指定筛选器
python3 scripts/run_all_screeners.py --date 2026-03-18 --screeners coffee_cup jin_feng_huang

# 快速模式(跳过新闻和LLM)
python3 scripts/run_all_screeners.py --date 2026-03-18 --no-news --no-llm

# 强制重新开始
python3 scripts/run_all_screeners.py --date 2026-03-18 --restart
```

### 3.3 Dashboard运行

```bash
# 1. 启动Dashboard
python3 dashboard/app.py

# 2. 浏览器访问
open http://localhost:5003

# 3. 选择筛选器并运行
# 在Web界面选择筛选器 → 选择日期 → 点击运行
```

## 5. 常见问题解答 (FAQ)

### Q1: 筛选器运行后没有结果，正常吗？

**A**: 正常。筛选器是**精确筛选**，不是每日都会产生结果。
- 咖啡杯形态: 平均每周1-3只
- 金凤凰: 平均每日0-5只
- 市场低迷期可能连续多日无结果

### Q2: 如何查看历史筛选结果？

**A**: 
```bash
# 方法1: 直接打开Excel
open data/screeners/coffee_cup/2026-03-15.xlsx

# 方法2: Dashboard查看
# 访问 http://localhost:5003 → 选择日期 → 查看结果

# 方法3: 命令行查询
ls -la data/screeners/coffee_cup/
```

### Q3: 筛选器运行报错"No module named 'xxx'"？

**A**: 安装缺失依赖:
```bash
pip3 install baostock pandas openpyxl requests
```

### Q4: 如何调整筛选器的敏感度？

**A**: 修改参数(以金凤凰为例):
```bash
# 更宽松(更多结果)
--max-pullback 0.05  # 允许5%回调
--min-days 2         # 最少2天横盘

# 更严格(更少结果)
--max-pullback 0.0   # 不允许回调
--min-days 4         # 最少4天横盘
```

### Q5: LLM分析很慢，可以关闭吗？

**A**: 可以，添加 `--no-llm` 参数:
```bash
python3 scripts/coffee_cup_screener.py --date 2026-03-18 --no-llm
```

### Q6: 如何生成筛选结果的图表？

**A**: 
```bash
# 生成咖啡杯图表
python3 scripts/plot_coffee_cup_charts.py --date 2026-03-18

# 生成组合图表(多只股票一页)
python3 scripts/plot_coffee_cup_charts.py --date 2026-03-18 --combined

# 限制生成数量
python3 scripts/plot_coffee_cup_charts.py --date 2026-03-18 --max-charts 20
```

### Q7: 筛选器支持盘中实时运行吗？

**A**: 部分支持:
- ** intraday_screener**: 盘中定时运行(09:35, 09:45, 10:00, 15:00)
- **postmarket_screener**: 收盘后运行(15:30)
- 其他筛选器: 建议盘后运行，基于日线数据

### Q8: 如何添加新的筛选器？

**A**: 
```bash
# 1. 继承base_screener.py
# 2. 实现run_screening方法
# 3. 放到scripts/目录
# 4. Dashboard会自动识别

# 参考示例:
cp scripts/base_screener.py scripts/my_screener.py
# 编辑 my_screener.py 实现筛选逻辑
```

### Q9: 筛选结果中的"置信度"是什么意思？

**A**: LLM分析给出的置信度分数(0-1):
- 0.8-1.0: 高置信度，形态清晰
- 0.6-0.8: 中等置信度，形态较清晰
- 0.4-0.6: 低置信度，形态模糊
- <0.4: 极低置信度，仅供参考

### Q10: 为什么有些股票被多次筛选到？

**A**: 正常现象，说明该股票同时符合多个形态:
- 咖啡杯 + 20日突破: 杯柄突破
- 金凤凰 + 试盘线: 多重确认
- 建议综合多个筛选器信号做决策
