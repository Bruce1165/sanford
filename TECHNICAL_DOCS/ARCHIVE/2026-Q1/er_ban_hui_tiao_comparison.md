# 二板回调筛选器 - 规则对比分析

## 对比来源

**用户规则（A-二板回调）** vs **现有代码** (`screeners/er_ban_hui_tiao_screener.py`)

---

## 用户规则描述

```
A-二板回调

1、时间范围：最近34个交易日。

2、信号一：出现连续2日的涨停板，并且第1个涨停板的成交额是前1日的成交额的 2倍以上。

3、信号二：在信号一的连续2日的涨停板后，任何1个交易日的最低价都不低于信号一的连续2日的涨停板的第1个涨停板的开盘价。

4、信号三：在信号一的连续2日的涨停板后，出现单日收涨且当日最高价格创连续两日涨停板以来的（每日最高价）新高时，即可确认为启动信号。
```

---

## 现有代码逻辑摘要

```python
# 核心参数
LIMIT_DAYS = 14  # 最近14个交易日

# 信号一：二连涨停板
- 连续2日涨幅≥9.9%
- 第1个涨停板成交额 ≥ 前1日成交额的2倍
- 第2个涨停日成交额 < 第1个涨停日成交额  # 额外条件

# 信号二：回调不破首板开盘价
- 二连板后任何交易日的最低价 ≥ 首板开盘价
- 在14天范围内
- 允许1%容忍度 (pullback_tolerance = 0.99)  # 额外参数

# 信号三：启动确认
- 单日收涨（涨幅>0）
- 阳线（收盘价 > 开盘价）  # 额外条件
- 成交额为T日（首板）以来的最大成交额  # 额外条件
- 最高价为T日以来的最高价
- 在14天范围内
```

| 条件 | 用户规则 | 现有代码实现 | 代码位置 | 状态 |
|------|----------|--------------|---------|------|
| 单日收涨 | ✓ | `current['pct_change'] <= 0` (return False if not) | Line 215 | ✅ 匹配 |
| 阳线（收盘价 > 开盘价） | ❌ 未提及 | `current['close'] <= current['open']` (return False if not) | Line 219 | ❌ **额外条件** |
| 成交额为T日以来的最大 | ❌ 未提及 | `current['amount'] < max_amount` (return False if not) | Line 229 | ❌ **额外条件** |
