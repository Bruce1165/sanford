## 技术栈概览

```
┌─────────────────────────────────────────────────────────────┐
│                        数据层                                │
│  NeoTrade2 SQLite (只读) + 本地 SQLite (读写)               │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        特征层                                │
│  技术指标 + 筛选器特征 + 市场相对特征                         │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        模型层                                │
│  XGBoost / LightGBM + 时间序列验证 + SHAP 解释                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        应用层                                │
│  预测服务 + Dashboard + 回测报告                              │
└─────────────────────────────────────────────────────────────┘
```
