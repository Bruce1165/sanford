# V4 咖啡杯柄筛选逻辑文档

## 概述

V4 版本咖啡杯柄筛选器采用 **5 轮迭代筛选**机制，逐步验证形态的各个组成部分。

---

## 形态结构

```
左侧杯沿 → 快速下调(5-60天) → 杯底震荡 → 快速上涨(5-25天) → 右侧杯沿 → 杯柄(0-13天)
```

---

## 筛选流程

### 第1轮：找右侧杯沿

**目的**：在筛选日期附近找到杯柄结束点

**操作**：
- 在筛选日期前 **13天**内搜索
- 找 **最高价** 作为右杯沿
- 计算杯柄长度 = 筛选日期索引 - 右杯沿索引

**通过条件**：
- 杯柄长度 ≤ 13天

---

### 第2轮：找左侧杯沿

**目的**：找到杯体开始的左侧杯沿点

**搜索范围**：
- 从右杯沿日期往前追溯
- 起始：**T-45**（离基准日最近的）
- 结束：**T-250**（最远的）
- 顺序：T-45 → T-46 → T-47 → ... → T-250

**通过条件**（三个条件全部满足）：

1. **价格匹配**：
   - 价格与右杯沿相差 ≤ 5%
   - 公式：`|左杯沿价 - 右杯沿价| / 右杯沿价 ≤ 0.05`

2. **局部高点**：
   - 左杯沿价格是其前后 **5天**内的最高价
   - 检查窗口：[左杯沿索引-5, 左杯沿索引+5]

3. **MA5趋势检查**（结果不影响筛选）：
   - 检查左杯沿前 **21天**的MA5是否呈上升趋势
   - 方法：将21天分成两半，比较后半部分均值 > 前半部分均值
   - 输出：在最终结果中标注 `ma5_passed` 和 `ma5_reason`

---

### 第3轮：验证杯深

**目的**：确保杯深在合理范围内

**杯深计算**：
```
杯深 = (左杯沿价格 - 杯底最低价) / 左杯沿价格
```

**通过条件**：
- 杯深范围：**5% - 70%**

---

### 第4轮：验证杯体结构、杯柄、成交量

**4.1 杯体结构验证**

**快速下调**：
- 天数范围：**5-60天**
- 从左杯沿开始下调

**震荡期**：
- 位于快速下调结束和快速上涨开始之间
- 价格上限：≤ 左杯沿价格 × 100%

**快速上涨**：
- 天数范围：**5-25天**
- 上涨到右杯沿

**4.2 杯柄验证**

**杯柄期间价格底线**：
```
安全水位 = 杯底价格 + 杯深 × 50%
杯柄期间最低价 ≥ 安全水位
```

**杯柄下跌限制**：
```
最大下跌 = 杯深 × 2.0
杯柄下跌 = (右杯沿价格 - 杯柄最低价) / 右杯沿价格
杯柄下跌 ≤ 最大下跌
```

**4.3 成交量验证**

**成交量对比**：
- 左侧杯沿后13天平均成交量
- 右侧杯沿前13天平均成交量

**通过条件**：
- 右杯沿前13天平均 / 左杯沿后13天平均 ≥ **2.0倍**

---

### 第5轮：当前价格验证

**目的**：确保当前价格没有跌破安全水位

**特殊处理**：
- 当杯柄长度 = 0（基点日 = 右杯沿）时，**跳过**此轮验证

**通过条件**：
- 当前价格 ≥ 杯底 + 杯深 × 50%

---

## 输出结果字段

| 字段 | 说明 |
|------|------|
| `code` | 股票代码 |
| `name` | 股票名称 |
| `close` | 筛选日收盘价 |
| `left_rim_date` | 左杯沿日期 |
| `left_rim_price` | 左杯沿价格 |
| `right_rim_date` | 右杯沿日期 |
| `right_rim_price` | 右杯沿价格 |
| `screening_date` | 筛选日期 |
| `screening_price` | 筛选价格 |
| `cup_body_width_days` | 杯体宽度（天数） |
| `cup_bottom_price` | 杯底价格 |
| `cup_depth_pct` | 杯深百分比 |
| `handle_days` | 杯柄天数 |
| `handle_low` | 杯柄最低价 |
| `handle_drop_pct` | 杯柄下跌百分比 |
| `price_diff_from_right_rim_pct` | 筛选价相对右杯沿涨跌 |
| `volume_ratio` | 成交量倍数 |
| `ma5_passed` | MA5趋势是否通过 |
| `ma5_reason` | MA5检查原因 |

---

## 参数配置

所有参数可在配置文件中调整：

| 参数组 | 参数名 | 默认值 | 说明 |
|--------|--------|--------|------|
| 杯沿相关 | `RIM_INTERVAL_MIN` | 45 | 杯沿间隔最小天数 |
| 杯沿相关 | `RIM_INTERVAL_MAX` | 250 | 杯沿间隔最大天数 |
| 杯沿相关 | `RIM_PRICE_MATCH_PCT` | 0.05 | 杯沿价格匹配度 ±5% |
| 杯沿相关 | `RIGHT_RIM_SEARCH_DAYS` | 13 | 右侧杯沿搜索天数 |
| 杯体相关 | `CUP_DEPTH_MIN` | 0.05 | 杯深最小 5% |
| 杯体相关 | `CUP_DEPTH_MAX` | 0.70 | 杯深最大 70% |
| 杯体相关 | `RAPID_DECLINE_MIN` | 5 | 快速下调最少天数 |
| 杯体相关 | `RAPID_DECLINE_MAX` | 60 | 快速下调最多天数 |
| 杯体相关 | `RAPID_ASCENT_MIN` | 5 | 快速上涨最少天数 |
| 杯体相关 | `RAPID_ASCENT_MAX` | 25 | 快速上涨最多天数 |
| 杯柄相关 | `HANDLE_MAX_DAYS` | 13 | 杯柄最大天数 |
| 杯柄相关 | `HANDLE_MAX_DROP_PCT` | 2.0 | 杯柄下跌占杯深比例 |
| 成交量相关 | `VOLUME_COMPARISON_DAYS` | 13 | 成交量对比天数 |
| 成交量相关 | `VOLUME_RATIO_THRESHOLD` | 2.0 | 成交量倍数要求 |
| 趋势相关 | `MA5_TREND_DAYS` | 14 | MA5趋势判断天数 |
| 趋势相关 | `OSCILLATION_PRICE_CEIL_PCT` | 1.0 | 震荡期价格上限占左杯沿% |
