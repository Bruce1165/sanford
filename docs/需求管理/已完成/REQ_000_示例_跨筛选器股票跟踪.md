## 1. 业务背景

**为什么要做这个功能？**

当前问题：
- 每个筛选器独立运行，结果互不关联
- 无法看到一只股票被多个筛选器识别的情况
- 无法对股票进行综合评分和排序
- 无法跟踪股票在不同筛选器中的表现变化

业务价值：
- 发现被多个筛选器同时识别的优质股票
- 建立统一的股票关注列表
- 提供量化评分，减少主观判断
- 跟踪股票随时间的变化趋势

## 2. 功能描述

### 2.1 输入
- 所有筛选器每日运行的结果（已有）
- 股票基础信息（已有）
- 每日行情数据（已有）

### 2.2 处理逻辑

**每日执行流程：**

```
步骤1: 触发筛选器运行
  - 获取所有筛选器的当日运行结果
  - 提取每只股票被哪些筛选器识别

步骤2: 计算股票的综合评分
  - 指标1: 形态强度 (权重 0.3)
    - 基于筛选器识别的形态质量
  - 指标2: 成交量突破 (权重 0.2)
    - 对比近5日平均成交量
  - 指标3: 板块动量 (权重 0.15)
    - 所在板块的整体涨跌幅
  - 指标4: 相对强度 (权重 0.15)
    - 相对大盘/板块的表现
  - 指标5: 时间衰减 (权重 0.2)
    - 识别后的天数衰减

步骤3: 板块对比分析
  - 计算同板块所有股票的平均评分
  - 计算该股票在板块中的排名
  - 计算相对板块的得分偏差

步骤4: 确定关注等级
  - overall_score >= 0.8: HIGH（高关注）
  - 0.5 <= overall_score < 0.8: MEDIUM（中关注）
  - overall_score < 0.5: LOW（低关注）

步骤5: 更新跟踪表
  - 记录每只股票的当日状态
  - 计算评分趋势（3日、7日变化）
```

### 2.3 输出
- 综合评分列表（所有股票）
- HIGH 级别关注列表（Top 50）
- 每只股票的详细评分信息
- 股票的趋势变化
- 展示形式：Monitor 页面的新标签页 + API

## 3. 判断条件与规则

### 3.1 判断条件
- **条件1：股票被 2+ 个筛选器识别**
  → 加权加成，形态强度指标额外 +0.1

- **条件2：连续 3 天被识别**
  → 时间衰减减少，形态强度指标额外 +0.05

- **条件3：板块涨跌幅 > 3%**
  → 板块动量指标 +0.1

- **条件4：成交量突破 2 倍**
  → 成交量突破指标 +0.2

### 3.2 评分规则

| 指标名称 | 权重 | 计算方式 | 阈值/等级 |
|---------|------|---------|----------|
| 形态强度 | 0.3 | 根据筛选器数量和质量打分，0-1 | >=0.8: 优秀 |
| 成交量突破 | 0.2 | 当日成交量 / 5日均值，归一化 0-1 | >=2.0: 突破 |
| 板块动量 | 0.15 | 板块当日涨跌幅，归一化 0-1 | >3%: 强 |
| 相对强度 | 0.15 | 股票涨幅 - 板块涨幅，归一化 0-1 | >5%: 强 |
| 时间衰减 | 0.2 | 识别后天数，指数衰减 1-0 | 当天: 1.0 |

### 3.3 时间相关规则
- **计算频率：** 每日收盘后自动执行
- **数据滞后：** T+1（次日计算前一日数据）
- **历史保留：** 保留所有历史数据，用于趋势分析
- **评分更新：** 每日重新计算所有股票的评分

## 6. 示例场景

**场景：**
- 日期：2026-04-03
- 股票：600519 贵州茅台
- 筛选器识别：咖啡杯形态、突破主升浪

**计算过程：**
1. 形态强度：被 2 个筛选器识别 → 0.8
2. 成交量突破：1.8 倍 → 0.8
3. 板块动量：白酒板块涨 2.5% → 0.7
4. 相对强度：涨 3.5%，板块涨 2.5% → 0.7
5. 时间衰减：识别当天 → 1.0

**加权计算：**
- overall_score = 0.8×0.3 + 0.8×0.2 + 0.7×0.15 + 0.7×0.15 + 1.0×0.2
- overall_score = 0.24 + 0.16 + 0.105 + 0.105 + 0.2 = **0.81**

**预期结果：**
- 关注等级：HIGH
- 综合评分：0.81
- 板块排名：白酒板块第 3 名
